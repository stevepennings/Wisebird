1. Make sure you have installed a version of Node.js supported by Polymer. To check the version of Node.js that you have installed, run:
		node --version

2. Update npm.
		npm install npm@latest -g

3. Ensure that Git is installed. (otherwise you can download the project from: https://github.com/stevepennings/wisebird.git, instead of cloning the repository)
		git --version

4. Install the latest version of Bower.
		npm install -g bower

5. Install Polymer CLI.
		npm install -g polymer-cli

6. Clone or download Wisebird.
		git clone https://github.com/stevepennings/wisebird.git

		Download link: 
		https://github.com/stevepennings/wisebird/archive/master.zip

7. Install NPM in project.
		npm install

7. Download necessary Webcomponents used in Wisebird.
		bower install

8. Run a local server.
		Polymer serve -o
